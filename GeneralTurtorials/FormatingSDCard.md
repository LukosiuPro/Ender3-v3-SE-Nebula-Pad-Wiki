# Formatting SD card

## Windows
1. Open file explorer
2. Find the sd card you have plugged in.
3. Right click on it.
4. Select format.
5. Now select the following options:
    - File System: **FAT32**
    - Allocation unit size: **4096 bytes**
    - Quick format: **checked**
6. Hit start.
